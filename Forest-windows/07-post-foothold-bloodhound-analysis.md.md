#ACCOUNTOPERATORS

The Windows Active Directory Account Operators group provides limited administrative privileges to manage user accounts and groups within a domain, making it a powerful but risky built-in group. Although it allows non-administrator users to create, modify, and delete most user accounts and groups, it cannot manage high-privilege groups such as Administrators, Domain Admins, or Server Operators, which helps reinforce security boundaries. Due to its capabilities, group membership should be strictly controlled and periodically audited.
